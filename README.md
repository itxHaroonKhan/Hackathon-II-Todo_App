# In-Memory Console Todo App

A simple, in-memory Python console todo application that allows users to manage their tasks without any persistence.

## Setup Instructions

1. Ensure you have Python 3.13+ installed on your system
2. Clone or download this repository
3. Navigate to the project directory
4. Run the application using Python:

```bash
python src/todo_app.py
```

## Features

- Add new tasks with titles and optional descriptions
- View all tasks with their completion status
- Update existing tasks
- Delete tasks
- Mark tasks as complete or incomplete
- Simple menu-driven interface

## Usage

The application provides a menu-driven interface where you can:
1. Add a new task
2. View all tasks
3. Update an existing task
4. Delete a task
5. Mark a task as complete/incomplete
6. Exit the application

All data is stored in-memory only and will be lost when the application exits.
